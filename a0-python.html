a0-python

  • Due Tuesday by 11:59pm
  • Points 50
  • Submitting an external tool

This assignment is to be completed individually! 

You must complete either this assignment of a0-R. 

The goals of the assignment are to ensure setting up the Python environment on the campus linux machines and the submission process with Gradescope. 

You will also be asked to start working with data in a pandas DataFrame. 

Instructions and information on the Python Setup are available: 

  • General Linux Resources - File Transfer
  • Python Setup (the first question in the assignment is to complete this process). 
  • Python Remote Access
  • Python Examples - After you get your system setup this is the page, I highly recommend returning to for each module.  It contains resources, references and example code for each topic we cover. 
    Look at these examples! 

Assignment files:  a0-python.files.zip Download a0-python.files.zip

When working with the files for this course, make sure in JupyterLab to select the "cs4821" or "[conda env:cs4821]" kernel that will use your Python environment you created. 

The assignment will have a mixture of autograded and manually graded questions (Q2).  The manually graded question will be graded after the due date.

Autograded questions can have a mixture of public and hidden test cases. Make sure you think carefully about what is being asked, you may pass the public test cases, but miss points because of the hidden cases. 

Do not change the file name, leave it as "a0.ipynb" for the autograder to work.
